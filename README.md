# Calculator (Java básico)

Compilar:
javac -d out src/Calculator.java

Ejecutar:
java -cp out Calculator add 2 3

java -cp out calculator add 2 3
java -cp out calculator sub 5 2
java -cp out calculator div 10 2

## Errores tipicos

-Ejecutar el programa fuer de la raiz del repositorio
-No compilar antes de ejecutar
